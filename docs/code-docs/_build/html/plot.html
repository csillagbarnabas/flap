flap.plot.sample_for_plot(x, y, x_error, y_error, n_points)[source]¶

Resamples the y(x) function to np points for plotting. This is useful for plotting large arrays in a way that short outlier pulses are still indicated. The original function is divided into np equal size blocks in x and in each block the minimum and maximum is determined. The output will contain 2*np number or points. Each consecutive point pair contains the minimum and maximum in a block, the time is the centre time of the block.

x: Input x array. y: input y array. np: Desired number of blocks. This would be a number larger than the number

of pixels in the plot in the horizontal direction.

Return values:

x_out, y_out If the box length is less than 5 the original data will be returned.
